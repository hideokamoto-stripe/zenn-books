## Webhook = If this, Then that 
* If this: もし〜が起きたら 
* Then that: その時〜を実行する 

###  Example 
* If this: サブスクリプションの支払いが完了したら 
* Then that: SFAに契約内容を同期する, 顧客にオンボーディングメールを送付する, 顧客にシステムへのアクセス権を付与する 47
